Tầng 3 — Location icons

Hiện tại Tầng 3 vi phạm rule "không emoji". Đây là demo set icon SVG thay thế.

▼ HIỆN TẠI — emoji + circle r=24

Emoji literal → render khác nhau giữa Windows / Mac / mobile. Vi phạm spec.

▲ ĐỀ XUẤT — <symbol> SVG icon

Icon SVG nhất quán mọi platform. Có shadow + dotted path nối location. Nền parchment. Label kèm chữ Hán nhỏ phía dưới — phù hợp tone cổ phong.
Kỹ thuật SVG dùng trong demo (đầy đủ)
  • feTurbulence + feColorMatrix → parchment grain (lớp np-grain baseFrequency 0.9 fine, lớp np-paper baseFrequency 0.013 large pattern).
  • feDisplacementMap → coastline wobble (lớp np-coast, áp lên polygon).
  • feMorphology + feGaussianBlur + feComposite → ink bleed cho stamps (np-ink).
  • linearGradient / radialGradient → mountain rock, snow, stamp ink (5 màu ink).
  • <symbol> + <use> → 1 lần define, dùng nhiều lần. Mountain 2 variant, forest cluster, cloud swirl, compass, 6 building icons.
  • pattern → ice cracks, ocean waves, ruins columns. Pattern fill nhẹ overlay lên polygon.
  • animate / animateTransform → player marker pulse (giữ subtle).
  • Layer order strict → parchment → aged spots → polygons → patterns → terrain symbols → islands → rivers → trade routes → stamps → labels → cloud border → vignette → compass → marker.
